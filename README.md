# Java Coursework

Java practice from my university bootcamp — core language fundamentals, OOP concepts, and array/DSA basics. This is a learning repo, not a finished project.

Source: university bootcamp coursework. A separate GFG (GeeksforGeeks) course section will be added later for additional DSA practice.

## Structure

- **`src/Basics/`** — control flow fundamentals: conditionals, loops, leap year check, tax calculation, misc practice
- **`src/OOPS/`** — OOP concepts: object creation, static variables, and factorial/Armstrong-number exercises built using classes
- **`src/Arrayprogram/`** — array & DSA basics: traversal, input handling, searching, sorting (bubble/insertion), rotation, insertion/deletion, frequency counting, uniqueness checks
